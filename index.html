Command Line Interface
The command line interface is a method to operate a computer using text commands. The method most people interact with their computers is the graphical user interface. With the graphical user interface, files and directories are represented visually, and the user is able to click on them with a mouse and drag them around. At the command line interface, the user does not see any visual representations, just texts.
Simple and
Powerful Tool

One of the reasons to use the command line interface is that your computer's graphical user interface may stop working. When this happens, you will have to use the command line interface. Another reason to use the command line interace is that you can perform various tasks faster at the command line interface than with the graphical user interface. It is also easier to automate tasks with scripts using the command line interface

The command line interface is accessed through the Command Prompt on a Windows computer. On the Mac, it is accessed through the Terminal. Although the command line interfaces of both operating systems have similar commands and syntax, there are important differences.
