The Command Prompt

You can access the Command Prompt by searching for "cmd" in the Start Menu. The commands that the Command Prompt uses are called DOS commands. Note that directory is another term for folder. Scroll down for a table of important DOS commands and click on the commands for further information.

Essential DOS Commands

Command Descripton
cd Change directory
cd.. Move to parent directory
cd\ Move to root directory
cls Clear the Command Prompt
copy Copy file
del Delete file
erase Delete file
format Erase entire drive
dir List files and folders
dir /ad List directories only
dir /p List files one page at a time
>dir /w List files and directories
horizontally with less information
md Make directory
move Move file
rd Remove Directory
ren Rename
driveletter: Move to specified drive
* Wildcard
