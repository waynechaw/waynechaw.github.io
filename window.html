The Command Prompt

You can access the Command Prompt by searching for "cmd" in the Start Menu. The commands that the Command Prompt uses are called DOS commands. Note that directory is another term for folder and capitalization does not matter. Scroll down for a table of important DOS commands and click on the commands for further information.

Essential DOS Commands

Command Descripton
cd Change directory
cd.. Move to parent directory
cd\ Move to root directory
cls Clear the Command Prompt
copy Copy file
del Delete file
erase Delete file
format Erase entire drive
dir List files and folders
dir /p List files one page at a time
dir /ad List directories only
dir /w List files and directories
horizontally with less information
md Make directory
move Move file
rd Remove Directory
ren Rename
driveletter: Move to specified drive
* Wildcard

cd

cd is the command used for changing directory. First type cd followed by the location of the folder you want to go to.

Syntax:

cd destination

In the example, I started at the root directory. I changed to the Users directory by typing cd user. Next, I changed to the "Wayne" directory under "Users" by typing cd Wayne. Finally, I demonstrated that you can change to a directory two levels below your current location by typing the parent directory first followed by the child directory such as desktop\yearup.

C:\>_

C:\>cd users

C:\Users>cd wayne

C:\Users\Wayne>cd desktop\yearup

C:\Users\Wayne\Desktop\yearup>_

cd..

cd.. is the command used for moving up a directory because .. represents the location of the parent directory of the command prompt's current location. Thus, cd.. changes command prompt's location to the parent directory.

Syntax:

cd..

In the example, I started at the directory:
C:\Users\Wayne\Desktop\yearup. Everytime I entered the command cd.. I moved up a directory until I got to the root directory.

C:\Users\Wayne\Desktop\yearup>cd..

C:\Users\Wayne\Desktop>cd..

C:\Users\Wayne>cd..

C:\Users>cd..

C:\>_

cd\

cd\ is the command used for moving to the root of the drive you are in.

Syntax:

cd\

In the example, I started at the directory:
C:\Users\Wayne\Desktop\yearup. Simply entering cd\ I moved to the root of the C drive

C:\Users\Wayne\Desktop\yearup>cd\

C:\>_

cls

If there is a lot of text on your command prompt and you want to clear it all, type cls to clear the screen.

Syntax:

cls

After you enter cls, you will see a screen similar to the example.

C:\>_

copy

The copy command allows you to duplicate a file and place the copy of the file in a different location.

Syntax:

copy source destination

My command prompt started at the root of C drive. I navigated to the Wayne directory by using the cd command. copy cmd.txt desktop copies the cmd.txt file from the Wayne directory to the desktop. Next, I copy the new file in desktop to the parent directory of Wayne, which is Users. It copied to Users because .. represents the parent of current location.

C:\>cd users

C:\Users>cd Wayne

C:\Users\Wayne>copy cmd.txt desktop

        1 file(s) copied.

C:\Users\Wayne>copy desktop\cmd.txt ..

        1 file(s) copied.

C:\Users\Wayne>_

del & erase

Both the del and erase commands delete file.

Syntax:

del source

Syntax:

erase source

In the example, I used both the del command and the erase command to delete the text files created in the copy example.

C:\>cd users

C:\Users>del cmd.txt

C:\Users>erase wayne\desktop\cmd.txt

C:\Users>_

format

The format command erases everything on the drive specified

Syntax:

format driveletter:

In the example, the command format d: would erase all data of my D drive

C:\>format d:

dir /w

The dir /w command displays only the name of each file and folder without other information in two columns.

Syntax:

dir /w source

I typed dir /w . after navigating to the "yearup" directory on my desktop to display the content of the directory in a short format. As you can see, the command prompt listed the items in two columns and does not show dates or file sizes of each item. Directories' names are in square brackets.

C:\>cd Users\Wayne\Desktop\yearup

C:\Users\Wayne\Desktop\yearup>dir /w .

  Volume Serial Number is C260-B1B8


Directory of C:\Users\Wayne\desktop\yearup

md

The md command creates a directory.

Syntax:

md destination

With the command prompt at the desktop directory, I type md firstfolder, I created a folder on my desktop called "firstfolder". Next, I enter md firstfolder\secondfolder to create another folder inside the folder I created.

C:\Users\Wayne\Desktop>md firstfolder

C:\Users\Wayne\Desktop>md firstfolder\secondfolder

move

The move command moves a directory or file.

Syntax:

move source destination

In the example, I moved the text file "grade.txt" from my desktop to the folder "myfolder" that is on my desktop. Next, I moved "grade.txt" to the parent directory of the desktop. Finally, I moved the folder from the desktop to the parent directory of the desktop.

C:\Users\Wayne\Desktop>move grade.txt myfolder
         1 file(s) moved.

C:\Users\Wayne\Desktop>move myfolder\grade.txt ..
         1 file(s) moved

C:\Users\Wayne\Desktop>move myfolder ..
         1 dir(s) moved

C:\Users\Wayne\Desktop>_

rd

The rd command deletes directory.

Syntax:

rd source

In the example, my command prompt's location is at the desktop. By typing rd emptyfolder, I remove the folder named emptyfolder on my desktop.

C:\Users\Wayne\Desktop>rd emptyfolder
C:\Users\Wayne\Desktop>_

ren

The ren command rename files or directories.

Syntax:

ren source newname

In the example, I renamed logo.png to icon.png and then renamed the oldfolder directory to newfolder.

C:\Users\Wayne\Desktop>ren logo.png icon.png
C:\Users\Wayne\Desktop>ren oldfolder newfolder
C:\Users\Wayne\Desktop>_

driveletter:

If there is another drive on your computer, you can change your command prompt's location to the root of that drive by typing the drive letter of the drive you want to change to followed by semicolon.

Syntax:

driveletter:

In the example, I jumped to the D drive by typing d: This will work no matter where the command prompt is currently at.

C:\Users\Wayne\Desktop>d:
D:\>_

*

* represents a wildcard. This means the * character can be used to substitute any character. With this technique, you can perform action on multiple files at once.

Syntax:

*

In the example, I moved all files from the "newfolder" directory to the desktop. Next, I delete all files with the .txt extention on my desktop.

C:\Users\Wayne\Desktop>move newfolder\*.* .
C:\Users\Wayne\Desktop>del *.txt
