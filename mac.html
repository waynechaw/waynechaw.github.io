The Terminal

You can access the Terminal by searching for "terminal" in the Spotlight search field. Scroll down for a table of important Terminal commands and click on the commands for further information.

cd

cd is the command used for changing directory. First type cd followed by the location of the folder you want to go to. You can also type cd without any arguments to change to the user's home directory.

Syntax:

cd destination

In the example, I started at the root directory. I changed to my home directory by typing cd. Next, I changed to my desktop my typing cd Desktop Finally, I print the current location of the terminal with the pwd command to confirm I am currently at my desktop in terminal.

$ cd

$ cd Desktop

$ pwd

/Users/Wayne/Desktop

cd ..

cd .. is the command used for changing to the parent directory of the current directory. This is because .. represents the location of the parent directory.

Syntax:

cd ..

In the example, I started at the desktop. I entered cd .. to move to the home directory.

$ pwd

/Users/Wayne/desktop

$ cd ..

$ pwd

/Users/Wayne

cd /

cd / is the command used for changing to the root directory.

Syntax:

cd /

In the example, I started at the desktop. I entered cd / to move to the root directory.

$ cd /

$ pwd

/

cd ~

cd ~ is the command used for changing to the user's home directory.

Syntax:

cd ~

In the example, I started at the root directory. I entered cd ~ to move to my home directory directory.

$ pwd

/

$ cd ~

$ pwd

/Users/Wayne

clear

clear is the command to clear the screen.

Syntax:

clear

After you enter clear, you will see a screen similar to the example.

$

cp

cp is the command to copy a file.

Syntax:

cp source destination

In the example, there is a picture named "mypic.png" on my desktop. I navigate to my desktop within the Terminal with the command cd ~/Desktop. Next, I copied the picture and named the duplicate "mypic2.png". Finally, I duplicated the picture again but placed it in my home directory.

$ cd ~/Desktop

$ cp mypic.png mypic2.png

$ cp mypic.png ../mypic3.png

diskutil list

diskutil list is the command to list all disks, their sizes and identifiers.

Syntax:

diskutil list

The example shows a sample output of diskutil list

$ diskutil list

 0:          EFI                209.7 MB     disk0

 1:    Apple_HFS  Machintosh    319.7 GB     disk0s1

 2:   Apple_Boot  Recovery      650.0 MB     disk0s2

ls

ls is the command to list all files and folders. You can add a location or simply type ls to display the content of current location.

Syntax:

ls source

In the example, I am currently at my desktop at the Terminal. I typed ls .. to list the items in my home directory. Next, I list the items in my desktop.

$ ls ..

Desktop       Downloads       Movies       Pictures
Documents     Library         Music        Public

$ ls

mypic.png    mypic2.png

ls -a

ls -a is the command to list all files and directories including hidden files and directories.

Syntax:

ls -a source

In the example, the hidden items have a period before them.

$ ls -a

.          .DS_Store       mypic2.png       mypic.png
..         .localized

$ ls

mypic.png    mypic2.png

ls -d */

ls -d */ is the command to list only directories.

Syntax:

ls -d */ source

In the example, use both ls -d */ and ls to demonstrate that ls -d */ only shows directories.

$ ls -d */

myfolder/

$ ls

mypic.png     mypic2.png     myfolder

mkdir

mkdir is the command to create a directory.

Syntax:

ls -d */ source

In the example, I navigated to my desktop and creates a folder named "images" on my desktop. Next, I created a folder named "files" in my home directory.

$ cd ~/Desktop

$ mkdir images

$ mkdir ../files

mount

mount is the command to mount a drive. To mount a drive means to make the drive map to the directory structure of your computer so you can access the content of the drive you are mounting. Drives are found in the dev folder under the root directory. You want to mount it to a folder inside Volumes or Media depending on your mac.

Syntax:

mount drive-name mount-location

In the example, I mounted my drive named disk0s4 to the directory myflashdrive. Make sure you create the directory first.

$ mount Dev/disks04 Volumes/myflashdrive

mv

The mv command is used to move files or directories to a different location.

Syntax:

mv source destination

In the example, I changed my location to my home directory. Next, I moved a directory named "files" to my desktop. Finally, I moved a picture on my desktop to my home directory. The period represents current location.

$ cd ~

$ mv files Desktop

$ mv Desktop/mypic.png .

pwd

The pwd command is used to print the Terminal's current location.

Syntax:

pwd

In the example, I used pwd to show me that I am in my home directory. I then navigate to my desktop and use pwd again to confirm I am now at my desktop.

$ pwd

/Users/Wayne

$ cd Desktop

$ pwd

/Users/Wayne/Desktop

rm

The rm command is used to remove file.

Syntax:

rm source

In the example, I removed "mypic2.png" and used the ls command to confirm.

$ ls

images     mypic2.png

$ rm mypic2.png

$ ls

images

rmdir

The rmdir command is used to remove directory.

Syntax:

rmdir source

In the example, I removed "images" and used the ls command to confirm. ls displays nothing because my directory is empty now.

$ ls

images

$ rmdir images

$ ls

rm -r

The rm -r command is used to remove directory and its content. This is the only way to delete a directory that contains subdirectories or files.

Syntax:

rm -r source

In the example, I removed the directory "images" which contains two pictures.

$ ls

image.png       image2.png

$ cd ..

$ rm -r images

*

* represents a wildcard. This means the * character can be used to substitute any character. With this technique, you can perform action on multiple files at once.

Syntax:

*

In the example, I removed all files starting with the letter b.

$ rm b*.*
