Scenarios

These are sample situations you may need to use the command line. Commands used for both the Command Prompt and the Terminal are given.

You want to delete all your pictures on your desktop.

Command Prompt
Terminal

C:\> cd \users\wayne\desktop
C:\Users\Wayne\Desktop>del *.jpg
C:\Users\Wayne\Desktop>del *.png

$ cd
$ cd Desktop
$ rm *.jpg
$ rm *.png

You want to move all Microsoft Word documents on desktop into a
folder named "documents" that is in your home directory.

Command Prompt
Terminal

C:\Users\Wayne\Desktop> move *.doc* ../documents

$ mv *.doc* ../documents

You want make a backup of all your files on your
desktop onto a USB drive named "mydrive".

Command Prompt
Terminal

C:\Users\Wayne\Desktop> copy *.* d:

$ cp *.* /Volumes/mydrive
